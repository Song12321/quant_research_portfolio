当前：无脑n->1合成
最优：
超级因子合成（Factor-Level）
→ 你把同类的“兄弟因子”合成成一个更强的“超级因子”。
例子：

ROE、毛利率、净利率 → 合成“质量因子”

PE、PB、股息率 → 合成“价值因子”

3/6/12 月动量 → 合成“动量因子”

MSCI 框架（Portfolio-Level）
→ 它拿这些“超级因子”来构建独立的组合，再组合这些组合。
例子：

用“质量因子”选股做一个 Quality Portfolio

用“价值因子”选股做一个 Value Portfolio

用“低波因子”选股做一个 Low Volatility Portfolio

然后 MSCI 指数 = 0.4×Quality + 0.3×Value + 0.3×Low Vol

详细参考：https://www.msci.com/eqb/methodology/meth_docs/MSCI_Diversified_Factor_Mix_Indexes_Methodology_Dec2014.pdf
